Как разместить на GitHub

Пошаговая инструкция:

  1. Создайте аккаунт на GitHub (если нет)
  2. Создайте новый репозиторий с названием username.github.io (где username - ваш логин)
  3. Загрузите этот HTML-файл в репозиторий, назвав его index.html
  4. В настройках репозитория перейдите в раздел "Pages"
  5. Выберите ветку main в качестве источника
  6. Через 1-2 минуты ваш сайт будет доступен по адресу: https://username.github.io

Альтернативный способ:

# Клонируйте репозиторий
git clone https://github.com/username/username.github.io

# Перейдите в папку
cd username.github.io

# Поместите файл index.html в папку
# Добавьте файлы в git
git add .

# Сделайте коммит
git commit -m "Добавлен сайт"

# Загрузите на GitHub
git push origin main
